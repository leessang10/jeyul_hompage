# Contact Reference Redesign Design

**Date:** 2026-04-08

**Goal**

현재 `contact` 상담 신청 페이지를 전면 폐기하고, 사용자가 제공한 레퍼런스와 최대한 동일한 구조와 인상으로 재구성한다. 브랜드 표기는 레퍼런스의 릴스퀘어 대신 모두 제율디앤씨 기준으로 교체한다.

**Context**

- 프로젝트는 Next.js App Router 기반이며 현재 상담 페이지는 [`app/contact/page.tsx`](/Users/isangmu/WebstormProjects/01_leessang10@naver.com/jeyul_hompage/app/contact/page.tsx)에 있다.
- 기존 페이지는 공용 `PageHero`, `SectionShell`, 카드형 정보 블록을 사용하지만, 이번 작업에서는 그 구조를 유지하지 않는다.
- 사용자는 데스크톱과 모바일 모두 첨부 레퍼런스에 최대한 동일한 구성을 원한다.

**Approved Direction**

1. 데스크톱은 좌측 대형 비주얼, 우측 폼 패널의 2단 분할 레이아웃으로 구성한다.
2. 모바일은 레퍼런스처럼 상단 이미지, 하단 폼의 세로 스택 레이아웃으로 전환한다.
3. 기존 상담 페이지의 모든 섹션과 카드, 보조 정보 영역은 제거한다.
4. 폼은 얇은 라인 중심의 미니멀 스타일로 전부 재구성한다.
5. 레퍼런스의 브랜드 관련 문구는 모두 제율디앤씨로 치환한다.
6. 우선순위는 실제 제출 기능보다 정적 UI 완성도와 시각적 일치도에 둔다.

**Information Architecture**

- 좌측 비주얼 영역
  - 좌상단 브랜드 로고 텍스트
  - 좌하단 대형 카피
  - 우하단 프로젝트명 텍스트
  - 우상단 닫기 아이콘 스타일 장식
- 우측 폼 영역
  - 페이지 제목 `인테리어 상담신청서`
  - 성함
  - 연락처
  - 주소
  - 평수
  - 가용 예산
  - 릴스퀘어를 알게 된 경로
  - 인테리어 예정일
  - 인테리어 플랜(선택)
  - 참고 사진(선택)
  - 개인정보처리방침 동의
  - 상담 신청하기 버튼

**Visual System**

- 배경은 화이트 또는 매우 옅은 웜 톤으로 통일한다.
- 폼 필드는 외곽 박스보다 하단 라인 중심으로 구성한다.
- CTA는 레퍼런스와 유사한 오렌지 필 버튼으로 처리한다.
- 좌측 이미지 오버레이 텍스트는 화이트 계열로 배치한다.
- 데스크톱에서는 페이지 전체 높이를 채우고, 모바일에서는 자연스러운 스크롤 흐름으로 전환한다.

**Behavior**

- 유입 경로는 버튼형 선택 칩으로 표현한다.
- 연락처는 세 개의 짧은 입력 구간으로 시각 분리한다.
- 사진 업로드는 정적 UI 기준의 업로드 박스로 표현한다.
- 닫기 아이콘은 시각적으로만 구현하고 별도 동작은 붙이지 않는다.

**Data / Content**

- 브랜드명, 주소, 전화번호 등은 기존 `siteContent`의 제율디앤씨 데이터를 우선 활용한다.
- 이미지 자산은 레퍼런스와 최대한 유사한 분위기를 만들 수 있도록 로컬 프로젝트에서 사용 가능한 방법으로 구현한다.
- 레퍼런스에 있던 릴스퀘어 표기는 모두 제율디앤씨로 치환한다.

**Testing / Verification**

- 본 프로젝트에 테스트 러너가 아직 없으므로 이번 작업에서는 TDD 기반 단위 테스트를 추가하지 않는다.
- 대신 정적 검증으로 `npm run lint`와 `npm run build`를 실행해 페이지 교체가 앱 전체 빌드를 깨지 않는지 확인한다.

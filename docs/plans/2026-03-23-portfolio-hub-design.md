# Portfolio Hub Design

**Date:** 2026-03-23

**Goal:** 기존 `/residential`, `/commercial` 소개형 페이지를 제거하고, 제율의 주거/기업 프로젝트를 하나의 아카이브 허브에서 필터로 탐색할 수 있는 그리드형 포트폴리오 구조로 재설계한다.

## Context

- 현재 사이트는 홈과 내부 페이지의 브랜드 톤은 정리되었지만, 포트폴리오 진입 구조는 여전히 `히어로 + 카피 + 설명 섹션` 중심이다.
- 사용자 요구는 레퍼런스 사이트처럼 프로젝트를 먼저 보여주는 아카이브형 UX다.
- 제율은 주거와 기업 프로젝트를 모두 수행하므로, 메뉴는 분리해도 실제 데이터와 탐색 경험은 통합 허브로 관리하는 편이 적합하다.

## Decision

### 1. 라우트 구조

- 신규 메인 허브: `/portfolio`
- 상세 페이지: `/portfolio/[slug]`
- 기존 메뉴 연결:
  - `주거 포트폴리오` -> `/portfolio?type=residential`
  - `기업·오피스 실적` -> `/portfolio?type=commercial`

### 2. 정보 구조

- 메뉴는 사용자 의도에 따라 분리한다.
- 데이터와 아카이브 화면은 하나의 `portfolio` 컬렉션으로 통합한다.
- 각 항목은 `type` 필드로 `residential | commercial` 구분한다.

### 3. 화면 구조

`/portfolio` 는 히어로 없이 바로 아카이브 헤더와 필터, 그리드로 시작한다.

상단 영역:
- 페이지 제목 `Portfolio`
- 짧은 설명
- 전체 개수 또는 현재 필터 개수

필터 영역:
- 1차 필터: `전체 / 주거 / 기업·오피스`
- 초기 버전은 1차 필터만 우선 적용
- 확장 시 2차 필터 추가 가능:
  - 주거: 평형대, 주거 유형
  - 기업: 오피스, 생산/연구시설, 리테일, 교육·공공

그리드 영역:
- 데스크탑 3열 기준
- 태블릿 2열
- 모바일 1열
- 카드 크기는 통일된 세로형 비율
- 카드 클릭 시 상세 페이지 이동

### 4. 카드 구성

공통:
- 썸네일
- 타입 배지
- 프로젝트명
- 메타 정보 1줄
- 짧은 설명 또는 범주 정보

주거 카드:
- 위치
- 평형
- 주거 유형

기업 카드:
- 클라이언트
- 시설 유형 또는 카테고리
- 위치

### 5. 기존 페이지 처리

- `/residential`, `/commercial` 페이지는 제거하지 않고, 필터 진입용 얇은 redirect 또는 lightweight wrapper 로 전환한다.
- 가장 단순한 방식은 `next/navigation` 의 redirect 로 `/portfolio?type=...` 로 보내는 것이다.
- 헤더 메뉴는 그대로 유지하되 목적지 URL 만 바꾼다.

## Visual Direction

- 레퍼런스처럼 프로젝트가 먼저 보이는 구조를 따른다.
- 다만 제율 톤을 유지하기 위해 완전한 게시판 UI가 아니라 편집형 아카이브로 정리한다.
- 상단 헤더는 현재 사이트의 투명/블러 전환 패턴과 별도로, 포트폴리오 화면에서는 처음부터 안정적인 밝은 헤더 상태가 자연스럽다.
- 카드 이미지는 실제 이미지가 없을 때도 기존 placeholder/ambient scene 계열을 재사용 가능하게 설계한다.

## Content Direction

- 설명은 최소화한다.
- 포트폴리오 허브에서는 브랜드 서사보다 탐색성이 우선이다.
- 헤드라인은 짧게 유지한다.
- CTA 문구는 상세 유입 또는 문의 유입으로 제한한다.

## Risks

- 현재 데이터 구조가 주거/기업으로 분리돼 있어, 통합 컬렉션 변환이 먼저 필요하다.
- 실제 이미지 자산이 없으면 아카이브의 설득력이 제한될 수 있다.
- `/residential`, `/commercial` 를 완전 삭제하면 기존 링크 호환성이 깨질 수 있으므로 redirect 처리해야 한다.

## Success Criteria

- `/portfolio` 에서 주거와 기업 프로젝트를 한 화면에서 탐색할 수 있다.
- `?type=` 쿼리로 필터 진입이 가능하다.
- 기존 `/residential`, `/commercial` 메뉴 사용자는 자연스럽게 통합 허브의 필터 상태로 진입한다.
- 페이지 첫 화면에서 프로젝트 카드 그리드가 바로 보인다.
